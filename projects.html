Isomorpher

Isomorpher was a project created for my undergraduate research assistant position in the Aggie Research Program. The research was on graph mining with the open source engine Peregrine. The objective was to find which isomorphic variant of a pattern would perform the best when mining a large social media graph and analyze what causes the differences in performace. It is written in C++ and allows you enter an input graph it will exhuastively generate all isomorphic variants of the pattern. It writes each variant in a seperate output file in the current directory and can addtionally generate LaTeX source code that displays each of the variants.

Point of Sales System

Point of Sales System was a group project in which we designed a point of sales system for a local buisnesses to track sales and manage invetory. It provided the resturant servers with an interface to select menu items and place orders, and provided the manager with an interface to track inventory, add menu items, and view sale reports data. The front-end was written in Java while the database was created using PostgreSQL and hosted on AWS.

Harmonia

Harmonia was created with team during HowdyHack 2021; a hackathon hosted by the organization TAMUHack at Texas A&M. All brainstorming, planning, coding, and testing was done withingin a 24 hour period. Because of this, we delegated functions among our team and incorporated code from each member into final product. It was written entirely in Python and utilized Tkinter for a graphical user interface to display virtual piano keyboard. Additionally it used the playsound module to allow user to learn about triads and harmonies through auditory juxtaposition.

Global Study Set

Global Study Set is a Java project that generates digital flashcards that automatically contains the same word in different languages. It allows user to study cards, take quizzes on them, and add their favorites to a collection to study later. It was heavily designed in object-oriented structure with advanced use of Java containers, generics, and bounded types. It utilizes a comma seperated text file containing parallel translation data to support five languages.

HACK Computer

The HACK Computer is a virutal general purpose computer built from the ground up by using only Nor gates by creating other logic gates with them. This was used to design various components of computer architecture such as Adders, Multiplexers, RAM, and CPUs. I also wrote an assembler in Python to convert an assembly language into machine code for this computer so that programs can be written for it. This project followed a structure from the course Nand2Tetris writen by Shimon Schocken and Noam Nisan.
