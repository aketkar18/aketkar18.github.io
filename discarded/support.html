Frequently Asked Questions

Do I need an account?

No. Discarded doesn't require any sign-up, login, or personal information.

How does Party Mode work?

A user creates a lobby and shares the code. Other players join using that code. One player joins the Knowers team and picks the secret card. Everyone else joins the Guessers and works together to ask questions and flip cards.

Does the Knower want the Guessers to win?

Yes. I originally designed this game so that the Knower loses if the Guessers guessed the card but I realized that makes no sense because the Knower is still obligated to answer truthfully. I've changed it so that Knower wins if the Guessers win, so they should be rooting for the success of the Guessers

Is AI Mode online?

Yes. AI Mode requires an internet connection because your questions are sent to Google's Gemini AI for answers. No personal information is included in these requests.

How do I report a player?

During a party mode game, you can report inappropriate messages by tapping and holding on it and using the report button. Reports include only the session ID, the player's nickname, the message, and a timestamp. All report data is automatically deleted after a short time.
